# 전략 가이드

## 📊 전략 개요

이 프로젝트는 **4개의 활성 전략**을 제공합니다. 각 전략은 특정 시장과 투자 성향에 맞게 최적화되어 있습니다.

---

## 🇺🇸 미국 시장 전략

### 1. 와인스타인 스테이지 (Weinstein Stage Analysis)

**대상 시장:** NASDAQ-100, S&P 500  
**투자 기간:** 중장기 (수주 ~ 수개월)  
**리스크:** 중간

#### 핵심 원리 (Weinstein)

- **Stage 2 (상승 추세)** 진입 포착
- 30주 이동평균선(MA30) 기준
- 시장 사이클 4단계 분석 (Stage 1~4)

#### 진입 조건

1. **Stage 2A 진입 (초기 돌파)**
   - 가격이 MA30 상향 돌파
   - MA30 우상향 중
   - 거래량 2배 이상 증가

2. **Stage 2 유지 (추세 확인)**
   - 가격 > MA30
   - MA30 우상향
   - 상대 강도 양호

#### 청산 조건

- 가격이 MA30 하회 (Stage 4 진입)
- MA30이 하향 전환
- 52주 고점 대비 -25% 하락

#### 장점

- 📈 명확한 추세 확인
- 🛡️ 중장기 안정적 수익
- 📊 시장 사이클 이해 가능

#### 단점

- ⏰ 진입 신호가 다소 늦을 수 있음
- 📉 횡보장에서 잦은 거짓 신호

---

### 2. 공격적 SEPA 2026 (Aggressive SEPA)

**대상 시장:** NASDAQ-100, 성장주  
**투자 기간:** 단중기 (수일 ~ 수주)  
**리스크:** 높음

#### 핵심 원리 (Aggressive SEPA)

- 미너비니 SEPA + 와인스타인 Stage 2 결합
- **트레일링 스탑**을 통한 수익 극대화
- 고변동성 시장(2025-2026) 최적화

#### 진입 조건 (STRIKE)

1. **트렌드 템플릿**
   - 50/150/200일 이동평균 정배열
   - 200일선 우상향
   - 52주 고점 근접 (25% 이내)

2. **VCP 패턴** (완화)
   - 변동성 수축 8% 이내
   - 거래 절벽 (30% 감소)

3. **피벗 돌파**
   - 고점 돌파
   - 거래량 1.5배 폭증

#### 청산 조건 (트레일링)

| 수익 구간 | 청산 조건 | 설명 |
| -------- | -------- | ---- |
| **50%+ (Tier 2)** | EMA10 이탈 | 슈퍼퍼포머 빠른 청산 |
| **15~50% (Tier 1)** | EMA20 이탈 | 추세 종료 감지 |
| **0~15% (초기)** | 8% 손절 | 초기 리스크 방어 |

#### 장점 (Aggressive SEPA)

- 🚀 슈퍼퍼포머 대형 수익 가능
- 💰 트레일링 스탑으로 수익 보호
- ⚡ 빠른 진입/청산

#### 단점 (Aggressive SEPA)

- 📊 엄격한 조건으로 신호 제한적
- 💥 고변동성 리스크
- ⏱️ 짧은 보유 기간

## 🇰🇷 한국 시장 전략

### 3. K-Minervini Pro (한국형 미너비니 프로)

**대상 시장:** KOSPI, KOSDAQ 대형주  
**투자 기간:** 중기 (수주 ~ 수개월)  
**리스크:** 중간

#### 핵심 원리 (K-Minervini Pro)

- 미너비니 SEPA의 한국 시장 최적화
- **240일선(1년선) 기관 매집** 확인
- 한국형 수급 폭발 (3.5배) 검증

#### 진입 조건 (K-STRIKE)

1. **한국형 트렌드 템플릿**
   - 50/120/240일 이동평균 정배열
   - 240일선 우상향 (기관 매집)
   - 52주 고점 -25% 이내

2. **한국형 VCP**
   - 변동성 수축 7% 이내 (더 타이트)
   - 거래 절벽 40% (평균 대비)
   - 최근 10거래일 수축

3. **한국형 피벗 돌파**
   - 피벗 가격 돌파
   - **거래량 3.5배 폭증** (한국 특화)

#### 청산 조건 (한국형 수익 보존)

| 수익 구간 | 청산 조건 | 설명 |
| -------- | -------- | ---- |
| **30%+ (급등)** | EMA10 이탈 | 테마주 전량 매도 |
| **10%+ (안정)** | 본절가(+2%) 하회 | 심리적 안정 |
| **0~10% (초기)** | 7% 손절 | 초기 리스크 방어 |

#### 장점 (K-Minervini Pro)

- 🏢 기관 매집 확인 (240일선)
- 📈 한국 시장 특성 반영
- 💎 대형주 안정성

#### 단점 (K-Minervini Pro)

- 📊 신호 발생 빈도 낮음
- ⏰ 240일 데이터 필요
- 🎯 대형주 위주 (중소형주 제외)

### 4. K-Weinstein (한국형 와인스타인)

**대상 시장:** KOSPI + KOSDAQ 전 종목  
**투자 기간:** 중기 (수주 ~ 수개월)  
**리스크:** 중간

#### 핵심 원리 (K-Weinstein)

- **EMA120 (120일 지수이동평균)** 기준
- Stage 2 초기 진입 포착
- 한국 전 종목 스크리닝

#### 진입 조건 (Stage 2)

1. **EMA120 돌파**
   - 전일 <= EMA120
   - 금일 > EMA120 (크로스)

2. **거래량 폭발**
   - 20일 평균 대비 **3배 이상**

3. **EMA120 우상향**
   - 상승 추세 확인

#### 청산 조건 (Stage 4)

- 가격이 EMA120 하향 돌파
- EMA120 하향 전환

#### 장점 (K-Weinstein)

- 🌏 전 종목 스크리닝 (2,000+ 종목)
- 🔍 숨은 강자 발굴 가능
- 📊 간단하고 명확한 규칙

#### 단점 (K-Weinstein)

- ⚠️ 거짓 돌파 가능성
- 📉 횡보장에서 비효율
- ⏱️ 전체 스크리닝 시간 소요 (5-10분)

## 🆘 도움말

### 신호가 없을 때

- 데이터 기간 확장 (6개월 → 12개월)
- 파라미터 완화 (VCP threshold 증가)
- 시장 상황 확인 (횡보장/하락장에서는 신호 감소)

### 너무 많은 신호

- 거래량 기준 강화 (1.5배 → 2.0배)
- VCP 수축 타이트하게 (8% → 6%)
- 상위 N개만 선택 (거래량/강도 순)
